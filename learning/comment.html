HTML Comments
HTML কোড এর বিভিন্ন অংশের ব্যাখ্যা করতে মূলত html comment ব্যাবহার করা হয় । html কমেন্ট ওয়েব পেইজে প্রদর্শিত হয় না। এটা শুধুমাত্র একটা কোডের ভিতরেই দেখা যায় এবং কোড বুঝতে সহায়তা করে। নিচের দেখানো উপায় মূলত কমেন্ট লেখা হয় এবং এই উপায়ে আমরা এক লাইনের কমেন্ট সহ বহু লাইনের কমেন্ট ও লিখতে পারবো :
<!--This is a comment-->

Comment Example 
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<!--This is a comment-->
<p>Comment is not show</p>

</body>
</html>

OUTPUT :

Comment is not show
