HTML Color
HTML কে সজ্জিত করার জন্য color এর ব্যাবহার ব্যাপক । আমরা color ছাড়া ওয়েব পেইজকে ভালোভাবে সজ্জিত করতে পারবো না। আমরা এখন নিচে পরপর উদাহরণের সাহায্যে যথাক্রমে rgb,hsl,hex,hsla,rgba এই color কোড গুলো উদাহরণ সহ বুঝবো । color কোড গুলো দেখুন আর নিজেরা প্র্যাকটিস করুন ।

Change background with rgb() 
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<div style="background-color: rgb(255,99,71);">This background color is changed by rgb.</div>
<br />
<div style="background-color: rgb(255,99,71);color:rgb(255,255,255);">This background and text color are changed by rgb.</div>

</body>
</html>

OUTPUT :
This background color is changed.

This background color and text color are changed.

rgb color code এ মূলত তিনটি value দিতে হয় <rgb(red , green, blue)> rgb লিখে প্রথমে red color এর পরিমান দিতে হয় এরপর green এর পরিমাণ এবং সবশেষে blue এর পরিমাণ । এবং সবচেয়ে গুরুত্তপূর্ণ কথা rgb color code এ red,green এবং blue er5 পরিমাণ 255 এর বেশি দেয়া যাবে না । যেমন : rgb(260,350,770) এমন লেখা যাবে নাহ । এদের সর্বোচ্চ value হবে 255 ।

Change background with color name 
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<div style="background-color: cyan;">This background color is changed by color name.</div>
<br />
<div style="background-color: magenta;color:white;">This background and text color are changed by color name.</div>

</body>
</html>

OUTPUT :
This background color is changed.

This background color and text color are changed.

উপরের উদাহরণে মূলত color এর নাম ব্যাবহার করে color change করা হয়েছে। তবে সকল ধরনের color নাম ব্যাবহার করে পাওয়া যায় না তাই এর solution হিসেবে color code গুলোকে ব্যাবহার করা হয় ।
Change background with hex and hsl color code 
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<div style="background-color: #3cb371;">This background color is changed by hex code.</div>
<br />
<div style="background-color: hsl(200,100%,50%);color:hsl(0,100%,100%);">This background and text color are changed by hsl code.</div>

</body>
</html>

OUTPUT :
This background color is changed.

This background color and text color are changed.

hex color code মূলত #rrggbb । এখানে প্রথমে red পরে green এবং সবশেষে blue এর hex কোড থাকে ।
hsl লেখার নিয়ম hsl( hue , saturation , lightness ) ।প্রথমে hue এরপর stauration এবং সবশেষে lightness থাকে এই তিনটির কম্বিশনে মূলত নতুন color create হয় ।
Color Alpha/Opacity 
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1 style="background-color:rgba(255, 99, 71, 0);">rgba(255, 99, 71, 0)</h1>
<h1 style="background-color:rgba(255, 99, 71, 0.2);">rgba(255, 99, 71, 0.2)</h1>
<h1 style="background-color:rgba(255, 99, 71, 0.4);">rgba(255, 99, 71, 0.4)</h1>
<h1 style="background-color:rgba(255, 99, 71, 0.6);">rgba(255, 99, 71, 0.6)</h1>
<h1 style="background-color:rgba(255, 99, 71, 0.8);">rgba(255, 99, 71, 0.8)</h1>
<h1 style="background-color:rgba(255, 99, 71, 1);">rgba(255, 99, 71, 1)</h1>

</body>
</html>

OUTPUT :

rgba(255, 99, 71, 0)

rgba(255, 99, 71, 0.2)

rgba(255, 99, 71, 0.4)

rgba(255, 99, 71, 0.6)

rgba(255, 99, 71, 0.8)

rgba(255, 99, 71, 1)


এই উদাহরণে মূলত opacity ব্যাবহার করা হয়েছে। সর্বোচ্চ opacity এর মান 1 এবং সর্বনিম্ন 0 । opacity মূলত রং এর গারত্ত। এক্ষেত্রে rgba তে একে alpha বলা হয়।
