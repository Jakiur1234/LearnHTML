Editor কি?
যার সাহায্যে টেক্সট এডিট করা এবং সেভ করা যায় তাকে এডিটর বলে ।
html শেখার জন্য লিখে রান করা প্রয়োজন। html code editor এর সাহায্যে কোড লিখে রান করা যাবে।
উইন্ডোজ এ নোটপ্যাড ব্যাবহার করে html কোড লেখা যায় । এছাড়াও আমাদের অ্যাপস এ বিদ্যমান কোড এডিটর এও html কোড লিখে রান করতে পারবেন।

কিছু জনপ্রিয় html এডিটর :
  • Notepad
  • Notepad ++
  • Vs Code

windows এ কোড লিখে রান করার উপায় :
Open Start > Programs > Accessories > Notepad
notepad চালু করার পূর্ববর্তী লেসনে দেখানো উপায়ে কোড লিখে ফেলুন ।
এরপর save as এ গিয়ে ফাইলটি সেভ করার সময় .txt এর বদলে .html ব্যাবহার করুন । এখন সেভ করা ফাইলটি ওপেন করলেই দেখবেন আপনি আপনার প্রথম কোড এর ভিউ দেখতে পাবেন ।
