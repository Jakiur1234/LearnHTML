HTML Tag
HTML elements বুঝতে হলে আমাদের প্রথমে html tag বুঝতে হবে । HTML introduction এ আমরা উদাহরণে যে <p>,<h1>,<,head>, <body> দেখেছি তার সবই html tag । প্রত্যেক html tag < এবং > এর মাধ্যমে ঘেরা থাকে । এক কথায় < এবং > দিয়ে ঘেরা যেকোনো অংশই html tag ।

Example of HTML Tag 
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>This is a heading tag.</h1>
<p>This is a paragraph tag.</p>

</body>
</html>

OUTPUT :

This is a heading tag.

This is a paragraph tag.


Example থেকে Tag নিয়ে আলোচনা
  • <!DOCTYPE html> থেকে শুরু করে <p> পর্যন্ত যতগুলো < এবং > দিয়ে ঘেরা অংশ রয়েছে তার সবই html tag । উদাহরণ : <html>,<body>,<head>,<title>,<h1>,<p> এর সবই ট্যাগ ।
  • এখানে লক্ষণীয় প্রত্যেক tag এর একটা start এবং end tag থাকে ।
  • কোনো ট্যাগ যে অংশ থেকে শুরু হয় সেটা start ট্যাগ । যেমন : <p>,<html> ইত্যাদি । এগুলো হলো start ট্যাগ ।
  • কোনো ট্যাগ যে অংশ থেকে শেষ হয় সেটাকে end tag বলে । যেমন : <p> এর end tag হলো </p>,<html> ট্যাগের end ট্যাগ হলো </html> ইত্যাদি ।

Empty and Nested Tag
যে সব ট্যাগের start ট্যাগ আছে কিন্তু end ট্যাগ নাই তাদের empty ট্যাগ বলে । যেমন : <br />,<hr />,<img /> ইত্যাদি ।

কোনো টাগের ভিতরে আরো একটা ট্যাগ থাকলে সেটাকে বলা হয় nested ট্যাগ । নিচের example এ nested ট্যাগ দেখানো হলো : 
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>
<!-----body tag nested in html tag.------->
<p>
This p tag is nested in body tag
<br />
<br />
<!----br is a empty tag---->
<span>
    This tag nested in paragraph tag
</span>
</p>
</body>
</html>

Note : উপরের উদাহরণে <!------ দিয়ে যে অংশগুলো আছে এর সবই মূলত কমেন্ট , কোড বোঝার জন্য আমরা কমেন্ট ব্যাবহার করি । পরে এ নিয়ে আলোচনা হবে । এখনই মাথা খারাপ করার প্রয়োজন নেই ।
OUTPUT :

This p tag is nested in body tag

This tag nested in paragraph tag


Example থেকে nested এবং empty ট্যাগের ব্যাখ্যা
  • <!DOCTYPE html> এবং <br /> উদাহরনে এই ট্যাগ দুইটা হলো empty ট্যাগ কারণ এদের start tag আছে কিন্তু end tag নেই ।
  • এখানে <html> ট্যাগের ভিতরে <body> ট্যাগ আছে । এখানে <body> ট্যাগ <html> ট্যাগের ভিতরে অর্থাৎ nested । এভাবে উদাহরণ থেকে বলা যায় <p> ট্যাগ <body> ট্যাগের ভিতর আবার <span> এবং <br /> ট্যাগ <p> ট্যাগের ভিতর nested হয়ে আছে ।
Fun : আমার মনে হয় না আপনার আর nested ট্যাগ বুঝতে প্রবলেম হবে !!! আপনি ধীরে ধীরে html এর রাজা হয়ে যাবেন !!!
HTML Elements
আমরা শিখেছি প্রত্যেক ট্যাগ এর start এবং end ট্যাগ থাকে । এই start এবং end ট্যাগের মাঝখানে অবস্থান করে element content । Element content এবং tag সহ পুরো অংশটিকে HTML Elements বলা হয় ।

Example of HTML Elements
<!DOCTYPE html>
<head>
<title>Page Title</title>
</head>
<body>
<h1>This is element content.</h1>
</body>
</html>

OUTPUT :

This is element content.


Example এর ব্যাখ্যা
  • start tag : <h1>.
  • element content : This is element content.
  • end tag : </h1>
  • তিনটিকে একত্রে বলা হচ্ছে HTML Elements.

Nested and Empty Elements
যে সকল html elements একটা element এর ভিতরে আরেকটা থাকে তাদের nested elements বলা হয় যা nested ট্যাগ এর মত একই । একটা ট্যাগের ভিতরে আরেকটা ট্যাগ থাকলে যেমন নেস্টেড ট্যাগ তেমনি element থাকলে সেটাকে nested element বলা হয় । আর empty HTML tag গুলোকেই empty HTML elements বলা হয় ।

Example of Empty and Nested HTML Elements 
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>
<p> 
    <b>
        Bold Text
    </b>
</p>
</body>
</html>

OUTPUT :

Bold Text


Example থেকে Empty এবং Nested HTML elements
  • এখানে <p> Element এর ভিতর <b>Bold Text </b> element আছে এইটা nested element.
    • List of Empty HTML elements :
    • <meta />
    • <br />
    • <hr />
    • <img />
