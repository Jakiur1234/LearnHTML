HTML Headings
HTML headings ট্যাগ মূলত কোনো ওয়েব পেইজের বা ডকুমেন্টের হেডিং কে নির্দেশ করে । Heading দিয়ে মূলত কোনো ওয়েব পেইজের মূল বক্তব্য নাম বা হেডলাইন লেখা হয় । html এ 6 ধরনের heading আছে । নিচের উদাহরন এ 6 ধরনের heading দেখানো হলো আশা করি এরপর আর হেডিং সক্রান্ত কোনো সমস্যা থাকবে না ।

Heading Example 
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

</body>
</html>

OUTPUT :

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6

Heading Style
যদিও প্রত্যেক heading এর একটি নির্দিষ্ট size আছে । তবুও আপনি চাইলে সেটা পরিবর্তন করতে পারবেন style attributes এর সাহায্যে । 
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>This is default h1</h1>
<h1 style="font-size : 100px;">This is 100px size h1</h1>

</body>
</html>

OUTPUT :

This is default h1

This is 100px size h1
