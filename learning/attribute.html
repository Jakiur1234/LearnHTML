HTML Attributes
HTML attributes মূলত html start tag এর সাথে অবস্থান করে এবং html elements সম্পর্কে অতিরিক্ত তথ্য প্রদান করে ।
Example of Attributes 
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<a href="https://www.google.com"target="blank">Go to google</a>
<br />
<img src="learning/images/intro-1.jpg"height="200px"width="200px"/>

</body>
</html>

OUTPUT :
Go to google

Example এর বর্ণনা
  • উপরের উদাহরণ থেকে স্পষ্টভাবে বোঝা যাচ্ছে যে কোনো ট্যাগ এর start tag এ attribute লিখতে হয় । attribute লেখার নিয়ম :
    • <start_tag atrribute_name = "value"></end_tag> (end tag তখনই হবে যখন tag টি empty নয় ।)
    • উপরের example এ <a> tag এর attribute হলো href এবং target যাদের value যথাক্রমে "link" এবং "_blank"
Note and Fun : আপনাকে এখন শুধু এতটুকু মনে রাখলেই চলবে যে attribute কিভাবে লিখতে হয় !! এখন থেকে ধীরে ধীরে আমরা অনেক অনেক tag এবং attribute দেখবো সাথে এদের ব্যাবহার ও দেখবো । আগেই ঘাবড়ানোর কিছু নেই href , target , src ইত্যাদি দেখে । যখন এগুলো শিখে ফেলবেন আপনি html এর রাজা হয়ে যাবেন ।

নিচে কিছু গুরুত্বপূর্ণ html tag এর ছক দেখানো হলো
মনোযোগ দিয়ে দেখুন কোনটার কি কাজ আমরা ধীরে ধীরে সব শিখবো । তবে আগে ভাগেই একটু জেনে রাখতে ক্ষতি কি?
