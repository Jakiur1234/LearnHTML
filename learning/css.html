CSS কি?

CSS এর পুর্ণরুপ হল Cascading Style Sheet. CSS এর সাহায্যে আপনি color, font, text size, margin, padding, background colors এইসব পরিবর্তন করতে পারবে এবং ওয়েবসাইট কে আরও সুন্দর এবং উন্নত করে গড়ে তুলতে পারবে ।

CSS এর সাহায্যে সকল element কে যেমন একসাথে ডিজাইন করতে পারবে ঠিক তেমন এই আলাদা আলাদা ভাবেও ডিজাইন করা যাবে । এতে সুবিধা হচ্ছে class এবং id ব্যবহার করে যেকোনো একটি নির্দিস্ট element কে অনেক সুন্দরভাবে ডিজাইন করা যায় ।
CSS এর ব্যবহারঃ
  • Inline CSS : HTML attribute এর মধ্যে style="" ব্যবহার করে "" এর ভিতরে CSS কোড লিখা হলে সেটাকে বলা হয় inline CSS.
  • Internal CSS : HTML এর <head> এর ভিতরে <style> ব্যবহার করে এর ভিতরে css কোড ব্যবহার করা হলে তাকে Inrernal CSS বলে ।
  • External CSS : মুল ডকুমেন্টের বাইরে .css এক্সটেনশন ব্যবহার করে ফাইল তৈরি করে মূল HTML ডকুমেন্ট এ link এর সাহায্যে যুক্ত করলে তাকে external CSS বলা হয় ।

Example এর সাহায্যে সকল CSS এর ধরন ব্যাখ্যা করা হলঃ

Example of inline CSS: 
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

    <div style="background-color: cyan;">এখানে background color inline css এর সাহায্যে পরিবর্তন করা হয়েছে । </div>

</body>
</html>

Inline CSS এর উদাহরণ এর বর্ননাঃ
  • এখানে div ট্যাগ এর ভিতরে <style> attribute এর মধ্যে যে ডিজাইন করা হয়েছে এটাকে inline css বলে ।

Internal CSS Example: 
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
<style>
    div{
        background-color: cyan;
    }
</style>
</head>
<body>

    <div>এখানে background color inline css এর সাহায্যে পরিবর্তন করা হয়েছে ।</div>

</body>
</html>

Internal CSS এর উদাহরণ এর বর্ননাঃ
  • এখানে <head> ট্যাগ এর ভিতরে <style> ট্যাগ ব্যবহার করে ডিজাইনের ব্যাকগ্রাউন্ড কালার পরিবর্তনের ব্যাপার টি ঘটেছে internal css এর কারনে । এখানে internal css মূলত <head> ট্যাগ এর মধ্যে <style> এর ভিতরে থাকে ।
