HTML কি?
  • HTML এর পূর্নরূপ হলো Hyper Text Makrup Language ।
  • Web page তৈরির জন্য HTML একটা আদর্শ markup language ।
  • HTML একটা ওয়েবসাইট এর কাঠামো তৈরি করতে ব্যাবহার করা হয় ।
  • HTML অনেকগুলো elements এর সমন্বয়ে গঠিত হয় ।
  • HTML elements গুলাই নির্ধারণ করে ব্রাউজার সকল কন্টেন্ট কিভাবে প্রদর্শিত হবে ।

Internal CSS এর উদাহরণ এর বর্ননাঃ
  • এখানে <head> ট্যাগ এর ভিতরে <style> ট্যাগ ব্যবহার করে ডিজাইনের ব্যাকগ্রাউন্ড কালার পরিবর্তনের ব্যাপার টি ঘটেছে internal css এর কারনে । এখানে internal css মূলত <head> ট্যাগ এর মধ্যে <style> এর ভিতরে থাকে ।

External CSS সম্পর্কে learn css অ্যাপস এ বিশদ জানতে পারবেন ।
